This folder includes templates and forms for contributing a task to the MIAT platform.  
Task contributions follow a two-stage process.

**1. Initial fit check:** 
Submit the completed *task fit checklist* (MIAT_task-fit-checklist.pdf), the *general task information* which would be used on the platform website (MIAT_task-information_website.pdf + task image), and the *peer-reviewed article* as a PDF or accessible link. A repository, README, and final technical testing are not required at this stage. The MIAT team reviews the task's fit and supporting evidence.

**2. Repository preparation and final review:** 
After preliminary acceptance, prepare the open-access repository, the README (MIAT_readme-template.md) and complete the *repository checklist* (MIAT_repository-checklist.pdf). Submit the completed checklist for final review. The task is added to the platform after the final review has been completed successfully.
